# Security

Soft Recall Demo is a static browser prototype.

## Expected Architecture

- Static Vite React app
- GitHub Pages deployment
- No backend
- No auth
- No database
- No project secrets required for normal local development

## Secrets

Do not commit secrets, API keys, runner tokens, deployment credentials, or private data.

If a workflow ever requires sensitive values, use GitHub repository secrets or environment secrets instead of committing them.

## Reporting

For security concerns, open a private report if GitHub security advisories are enabled for the repository. Otherwise, contact the repository owner directly with a concise description and reproduction steps.
