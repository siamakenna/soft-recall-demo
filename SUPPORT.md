# Support

Soft Recall Demo support is focused on gameplay and deployment issues.

## Reporting Issues

When reporting a problem, include:

- browser and version
- operating system
- URL tested
- route taken through the demo
- expected behavior
- actual behavior
- screenshots or video, if helpful
- browser console errors, if available

Useful route format:

```text
Title -> Begin -> Bedroom -> Phone -> Hallway -> ...
```

## Live Demo

```text

```

## Scope

This project is PC-only / desktop-first for now. Mobile/PWA requests, audio requests, backend requests, auth requests, and database requests are out of scope for the current demo.
